
/* Read Me */

This package for setting up Animal Roar Sounds of NPC.

So that, You can set up AudioSource Component.


  - Setup Roar Sound for NPC -
  
  1) Select NPC in Hierarchy or Project Fold.
  
  2) Click "Add Component Button " of Inspector and Add "AudioSource"
  
  3) Drag and Drop to Audio Clip with Audio Clips ("BearAttacksnd, StagRoarSnd, TigerAttack Snd... etc") of "00.Sounds."
